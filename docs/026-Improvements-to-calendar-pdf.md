# Updates to the PDF Calendar Format

## Header Section

Add a line in the header between the first one "Presentation Council..." and the second one "Council Meetings are...": this new line will have the address of the Council all in one line (271 West Saddle River Road, Upper Saddle River, NJ 07458).
The second line (now third line) will be spaced from the first two, and centered horizontally on the line. Keep a small empty space also between the third line and the gold divider line, so that the whole title is clearly separated from the following contents

## Calendar Months

The progression of the months needs to be vertical (the left column is filled first, then the second column, in order by increasing month). To avoid fillng the first column too much as compared to the second one, break to the second column after the month of December

Align the dates to the right of their space and add a hyphen between the date and the event title on each line, because it's hard to match dates and events when they are so packed together.
For events of type officers_meeting, replace the hyphen with a small diamond shaped symbol, and use a Navy blue font color for the entire line (date and Event title).
For events of type council_meeting, replace the hyphen with an asterisk, and use a bold, red font for the entire line (date and Event title).
